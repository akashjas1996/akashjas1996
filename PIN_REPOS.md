# Pin these repos on your GitHub profile

GitHub doesn't expose a public API for profile pins — do this once in the browser (~30 seconds):

1. Open https://github.com/akashjas1996?tab=repositories
2. Click **Customize your pins**
3. Select:
   - `Wooble-org/wooble-company`
   - `Wooble-org/wooble-engine`
   - `Wooble-org/wooble-android`
4. Click **Save pins**
